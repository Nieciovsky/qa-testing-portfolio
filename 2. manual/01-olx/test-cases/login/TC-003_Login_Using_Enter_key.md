# TC-003 – Login by pressing Enter key instead of button

**Project:** OLX.pl  
**Module:** Authentication → Login form  
**Type:** Positive – Accessibility / Usability  
**Priority:** Medium  
**Preconditions:** Form is filled with valid data

**Test Data:** Same as TC-001

**Steps to Reproduce:**
1. Navigate to login page (as in TC-001 steps 1–5)
2. After entering valid email and password, press the **Enter** key on the keyboard (instead of clicking the button)

**Expected Result:**
- Login is successful (same as clicking the button)
- Redirect to "Moje OLX"
- No errors

**Actual Result:** [Same as expected – success]

**Status:** ✅ Passed
